# the-beatdown — Vision

A competitive web music sequencer. Compose within strict constraints, battle head-to-head in the Arena.

## Scope guardrails

- Single-player V1 for Sprints 1–2; Arena/competition in Sprint 3
- Desktop-first (mobile deferred)
- Local models for bulk work; cloud only when escalated (Gemma4:31b-cloud → Gemini Flash)

## Sprints

- **Sprint 1** ✅ — Mondrian shell + 5-module Web Audio API engine
- **Sprint 2** ✅ — Vault save/load, arrangement timeline, drag/snap/delete
- **Sprint 3** 🔲 — Supabase, Auth, Arena, Gatekeeper, Matchmaker AI, ELO leaderboard

## Not in V1

- Mobile layout
- Real drum synthesis (noise/transient-based kicks/snares)
- Pattern rename UI
- Export / audio render
- Collaboration / multiplayer editing
